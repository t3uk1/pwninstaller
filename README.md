# Pwninstaller Demo

This is a script that can install basic pwn environment.
There may be some bugs, please understand :)

## Install list
* peda
* Pwngdb
* pwndbg
* perl
* zstd
* gcc
* patchelf
* libc6-dev-i386
* one_gadget
* seccomp-tools
* pwntools
* LibcSearcher
* ROPgadget
* ropper

## How to use?
clone this repo
```git clone https://github.com/t3uk1/pwninstaller.git```
then enter ./pwninstaller
```cd ./pwninstaller```
run pwninstaller.sh
```./pwninstaller.sh```
